Getting Started

This tutorial shows you how to get started with our API.

curl -X GET \
  https://api.example.com/balance \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json'

Transferring Assets

This tutorial shows you how to transfer assets between users.

curl -X POST \
  https://api.example.com/transfer \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{"from": "user1", "to": "user2", "amount": 10}'
